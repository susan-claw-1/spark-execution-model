# Spark Execution Model

A short, sharp learning plan to understand Apache Spark's execution model from the inside out.

## Scope

- **Language:** PySpark / Spark SQL
- **Runtime:** Databricks (generic — runtime-agnostic)
- **Delta Lake:** Yes — data lake considerations included
- **Duration:** ~2 weeks, daily bite-sized lessons delivered via chat

## Structure

```
labs/          — day-by-day PySpark labs (runnable in Databricks notebooks)
docs/          — concept deep-dives and architecture notes
progress/      — learning plan checklist and progress tracker
configs/       — sample Spark configurations
data/          — small demo datasets for labs
notebooks/     — optional .ipynb exports
src/           — reference material organised by topic
```

## How It Works

Lessons are delivered conversationally. Each day covers a concept with a short explanation, a runnable PySpark snippet, and follow-up discussion. Labs are committed here for reference.
